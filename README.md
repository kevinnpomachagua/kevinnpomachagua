# Hola, soy Kevinn Pomachagua

**Analista de Aseguramiento y Control de Calidad de Software (QA | QC)** con más de 5 años de experiencia liderando la validación de sistemas críticos de alto impacto, tanto en el sector público como privado. 

Me especializo en el ciclo completo de pruebas (**STLC**), desde el análisis funcional y diseño de casos de prueba hasta el despliegue de estrategias avanzadas de automatización bajo metodologías ágiles.

- 🎯 **Enfoque actual:** Automatización de pruebas End-to-End (E2E), API testing y mejora continua de procesos QA.
- 💼 **Último rol:** Analista de Calidad (QA) en el Organismo Especializado para las Contrataciones Públicas Eficientes.
